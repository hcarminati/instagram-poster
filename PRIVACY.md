# Privacy Policy for Pisces Meme Poster

Last updated: December 29, 2024

## Overview
Pisces Meme Poster is an automated Instagram posting application that generates and posts meme content to Instagram.

## Information We Collect
- Instagram Business Account access (to post content)
- No personal user data is collected or stored

## How We Use Information
- To generate and post meme images to your Instagram account
- To access Instagram API for content publishing

## Data Storage
- Access tokens are stored securely in GitHub Secrets
- Generated images are temporarily hosted on ImgBB
- No user data is permanently stored

## Third-Party Services
- Instagram Graph API (Meta)
- ImgBB (image hosting)
